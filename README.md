# Internal Medicine Morning Report Presentation
## Learning From Complex Cases: Diagnostic Reasoning in Hospital Medicine

### Overview
This interactive presentation explores diagnostic reasoning, comprehensive chart review, and care coordination through a real de-identified clinical case. The presentation is optimized for iPad viewing and emphasizes learning and systems improvement.

### Features
- ✅ **iPad-Optimized:** Touch-friendly navigation, responsive design
- ✅ **Interactive:** Keyboard and swipe navigation
- ✅ **Professional:** Clean, medical education-focused design
- ✅ **Self-Contained:** Single HTML file, no dependencies
- ✅ **Print-Friendly:** Formatted for PDF export if needed

### How to Use

#### On GitHub Pages:
1. Visit: `https://[your-username].github.io/[repository-name]/`
2. Navigate using:
   - **Touch:** Swipe left/right on iPad
   - **Keyboard:** Arrow keys (← →)
   - **Buttons:** Click Previous/Next buttons
3. View slide counter in top-right corner

#### Local Viewing:
1. Download `index.html`
2. Open in any modern web browser
3. Works offline - no internet required

### Learning Objectives
- Develop systematic approaches to comprehensive chart review
- Recognize infection risk in chronic lymphoproliferative disorders
- Apply clinical reasoning when laboratory and clinical findings diverge
- Implement antibiotic stewardship principles
- Identify system-level opportunities for quality improvement

### Presentation Structure
24 slides covering:
1. Case Introduction
2. Clinical Presentation & Workup
3. Diagnostic Reasoning
4. Learning Opportunities
5. Teaching Points (Chart Review, CLL, Stewardship, Care Coordination)
6. Systems Thinking & Quality Improvement

### Privacy & Ethics
- All identifying information removed
- Case presented in educational, non-judgmental framework
- Focus on learning and systems improvement
- HIPAA-compliant presentation

### Technical Requirements
- Modern web browser (Safari, Chrome, Firefox, Edge)
